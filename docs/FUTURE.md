# EverythingBeer Future Enhancements

EverythingBeer aims to become the definitive platform for beer discovery, education, and community. This document outlines potential future enhancements beyond the current roadmap.

  - Web scraping (with permission)

- Current beer availability
- Seasonal release tracking
- New beer announcements

### Historical Beer Data
- Discontinued beers archive
- Vintage beer information
